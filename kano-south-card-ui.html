Design Notes

  • Format: Each card uses a 9:16 vertical ratio suitable for WhatsApp Status, Facebook Stories, TikTok covers and Telegram.
  • Brand colors: Emerald #006633, mint #E6F2ED, charcoal #222222, gold #D9A441.
  • Copy discipline: The designs ask for documents and public proof. They avoid legally risky accusations.
  • Export: Open this file in Chrome, zoom to the desired card, screenshot/export at 1080×1920. You can also print to PDF; each card is prepared for page breaks in print mode.
